# Gesture Recognition Color Matching Game

A real-time interactive game that uses hand gesture recognition to match colored ball pairs.

## Features
- Real-time hand detection with MediaPipe
- Match same-colored ball pairs using gestures
- Line drawing visualization
- Combo multiplier system
- Progressive difficulty levels

## How to Play
1. Move hand near a colored ball (it highlights)
2. Move to another ball of SAME COLOR
3. When colors match, they connect with a line
4. Match all pairs to advance level
5. Build combos for higher scores

## Requirements
- Python 3.11+
- Webcam
- opencv-python, mediapipe, numpy
